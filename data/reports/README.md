# Reports Directory

This directory stores exported reports generated by the Hospital Management System.

## Purpose

When you use the `ReportGenerator` service to generate and export reports, the output files will be saved here.

## Supported Export Formats

The ReportGenerator supports three export formats:

1. **Text (.txt)** - Plain text format, easy to read in any text editor
2. **CSV (.csv)** - Comma-separated values, can be opened in Excel or spreadsheet applications
3. **HTML (.html)** - HTML format, can be opened in web browsers with formatted tables

## Report Types

The system can generate the following types of reports:

- **Daily Reports** - Appointments and revenue for a specific date
- **Weekly Reports** - Summary of a week's activities
- **Monthly Reports** - Monthly statistics and summaries
- **Revenue Reports** - Financial reports for a date range
- **Patient Reports** - Patient statistics and demographics
- **Doctor Performance Reports** - Doctor metrics and appointment counts
- **Appointment Analysis** - Breakdown of appointments by status

## Example Usage

```cpp
using namespace HMS::BLL;

// Get the ReportGenerator instance
auto* reportGen = ReportGenerator::getInstance();

// Generate a daily report
auto report = reportGen->generateDailyReport("2024-12-31");

// Export to different formats
reportGen->exportReport(report, ExportFormat::TEXT, "data/reports/daily_2024-12-31.txt");
reportGen->exportReport(report, ExportFormat::CSV, "data/reports/daily_2024-12-31.csv");
reportGen->exportReport(report, ExportFormat::HTML, "data/reports/daily_2024-12-31.html");
```

## File Naming Convention

Recommended naming pattern:
- Daily: `daily_YYYY-MM-DD.{txt|csv|html}`
- Weekly: `weekly_YYYY-MM-DD_to_YYYY-MM-DD.{txt|csv|html}`
- Monthly: `monthly_YYYY-MM.{txt|csv|html}`
- Revenue: `revenue_YYYY-MM-DD_to_YYYY-MM-DD.{txt|csv|html}`
- Patient: `patient_report_YYYY-MM-DD.{txt|csv|html}`
- Doctor: `doctor_performance_YYYY-MM-DD.{txt|csv|html}`

## Notes

- Reports are automatically timestamped
- Old reports are not automatically deleted - manage this directory periodically
- CSV reports can be imported into data analysis tools
- HTML reports include formatted tables for better readability
